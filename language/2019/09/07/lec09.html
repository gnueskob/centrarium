C++ 01 - 표현식과 문장 (1)

by Gnues

2019. 9. 7. 18:42
in Language

그간 사용해오던 언어들을 다시 정리하고 애매모호한 내용들을 바로잡기 위한 글입니다.

표현식

모든 변수의 이름, 상수, 리터럴은 표현식이다.

x, y

연산자로 결합한 하나 이상의 표현식들은 표현식을 구성한다.

x + y, x * y + z

또한 할당문도, 입출력 작업도 표현식이다. 표현식을 다른 할당문 안에 사용할 수 있다.

x = y + z, x = y = z + 1, std::cout << "x is " << x << std::endl

표현식을 인자로 넣은 함수 호출도 마찬가지로 표현식이다.

abs(x), abs(x * y + z), pow(abs(x), y)

문장

끝에 세미콜론(;)이 오는 표현식들이 문장이다.

x = y + z;
y = f(x + z) * 3.5;
x + y;

마지막 x + y;도 문장이지만 계산한뒤 값을 사용하지 않고 날려버린다.

최신 컴파일러는 이런 쓸모없는 계산도 최적화해주기도 한다.

만약 사용자 정의 타입의 개체로 덧셈등을 직접 정의해서 사용한다면 쓸모없는 계산이 아니게 될 수도 있다.

하나의 세미콜론은 빈 문장이며 세미클론만으로도 유효한 문장 하나이다.

함수 정의같은 복합문은 세미콜론으로 끝나지 않는다.
